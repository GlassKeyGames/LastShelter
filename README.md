# Last Shelter

**Last Shelter** is an Unreal Engine 5.4 C++ zombie survival shooter where the player must fight zombies, collect resources, build shelter pieces, and complete step-based objectives to survive.

The game focuses on survival gameplay, resource collection, combat, and small puzzle-style progression tasks. Players use tools like an axe and pickaxe to collect wood and stone, while also managing threats from zombies throughout the level.

## Features

* Unreal Engine 5.4 project
* C++ gameplay systems
* Zombie shooter combat
* Wood and stone resource collection
* Axe and pickaxe tool usage
* Step-based objectives and puzzle progression
* Player HUD and interaction prompts
* Build/shelter survival mechanics
* Weapon and tool-based gameplay
* Enemy AI encounters

## Gameplay Overview

The player explores the environment, collects resources, fights zombies, and completes objectives to progress. Wood and stone are collected using tools, and the player must survive while moving through different gameplay steps and challenges.

## Repository Notes

This project uses **Git LFS** for Unreal Engine assets such as `.uasset` and `.umap` files.

Packaged builds are not stored in this repository. Playable builds should be uploaded separately to a platform such as itch.io or cloud storage.

## How to Open

1. Clone the repository.
2. Make sure Git LFS is installed.
3. Open `LastShelter.uproject` with Unreal Engine 5.4.
4. Let Unreal rebuild project files if needed.
5. Open the project and play from the main test level.

## Git LFS

If cloning this project, run:

```bash
git lfs install
git lfs pull
```
